## AudioComics

Веб-приложение для прослушивания комиксов с аудио-озвучкой.
Проект ориентирован на **людей с нарушениями зрения** и использует **audio-first интерфейс** с учётом требований доступности (ARIA, screen reader best practices).

## Как запустить проект локально

### Склонировать репозиторий

```bash
git clone https://github.com/tadaro-2/megaschool_accessibility.git
cd megaschool_accessibility
```

### Установить зависимости

```bash
npm install
```

### Запустить dev-сервер

```bash
npm run dev
```

### Открыть сайт в браузере

После запуска в терминале появится адрес, например:

```text
http://localhost:5173/
```
